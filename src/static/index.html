How It Works

This website uses IP-based geolocation to determine your location, including street-level information when available. The data is stored in a cookie on your device for 30 days.

The following information is collected:

  • IP Address
  • City
  • Region
  • Country
  • Postal Code
  • Geographic Coordinates
  • Street Information (when available)

You can clear this information at any time by clicking the "Clear Location Cookie" button.
